Protocol Mechanics
Attention Oracle
Weighted signals from on-chain txs, staking, social mentions, API calls. 7-day rolling score with 3.5-day half-life decay.
L0: DATA
Energy Pool (VRDE)
Logarithmic bonding curve converts attention to soulbound energy. Non-transferable token with 90-day expiry. Anti-monopoly by design.
L1: ENERGY
Agent DNA
On-chain genome: skills, memory, behavior, reputation, mutation seed. NFT with composable metadata schema.
L2: GENOME
Reproduction Engine
Threshold-gated spawning. Burns 100 VRDE per parent. Blends DNA with VRF-driven mutation. Mint new agent NFT. 14-day cooldown. Max 3 per quarter.
L3: LIFE
Anti-Gaming
